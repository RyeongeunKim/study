form

  • form 태그란? 사용자가 서버에 정보를 쉽게 전달할 수 있는 페이지를 구성함
  • form 태그 속성
    • action: 서버의 파일 지정
    • method: 전달 방식 지정 -> 일반적으로는 post, 보안상의 문제가 없고 길이가 짧을 때는 get

input type="text" 한줄 글상자

  • label 태그: input상자에 입력할 설명문 지정
  • for의 속성값을 input태그의 id속성값과 일치시켜 글상자와 설명문 연결시킴
  • size 속성: 상자의 가로 길이 지정
  • maxlength 속성: 입력데이터의 최대길이 제한
  • minlength 속성: 입력데이터의 최소길이 제한

input type="search" 검색 상자(삭제버튼 생성)

input type="hidden" 숨긴 데이터(사용자에게 보이지 않는 데이터 전송용)

input type="file" 파일 전송용

input type="submit/reset/button" 전송/초기화/일반버튼

  • submit: 폼의 정보 전송
  • reset: 초기화
  • button: 폼과는 무관한 일반 버튼
  • value 속성: 버튼의 글자

input type="image" 전송하는 역할의 그림

src 속성으로 그림파일의 경로를 지정

button ... /button 전송/초기화/일반 버튼

  • button은 기본적으로 폼의 정보를 전송하는 역할임
  • type="reset" : 폼의 정보 초기화
  • type="button" : 폼과는 무관한 일반 버튼
  • 버튼의 글자는 시작과 종료 태그 사이에 입력
