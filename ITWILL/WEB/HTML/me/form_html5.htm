html5에서 추가된 폼요소들

input type="OOO" OOO에 넣어서 사용!

[날짜와 시간]
  • time : 시간(시분)

  • date : 날짜(연도-월-일)

  • datetime-local : 날짜와 시간

  • week : 일주일 단위

  • month : 한달 단위

[개인정보]
  • tel : 전화번호(모바일에서 가상키패드가 숫자형식으로 바뀔 수 있고 전화걸기 가능)

  • email : 이메일주소
    모바일에서 가상키패드가 영문(.com) 포함하는 형식으로 바뀔 수 있음
    브라우저에서 알맞은 이메일 형식인지(@여부, 뒷자리) 체크

  • url : 웹사이트 주소 지정
    모바일에서 가상키패드가 영문(https://) 포함하는 형식으로 바뀔 수 있음
    브라우저에서 알맞은 주소 형식인지(https:// 등 프로토콜 여부) 체크

[숫자/컬러]
  • number : 숫자값 지정, 증감버튼 생성

    • min : 최소값(0 지정)

    • max : 최대값(20 지정)

    • step : 증감 간격(12 지정)

  • range : 시각적으로 범위 내의 값 지정

  • color : 컬러값 지정

html5에서 추가된 속성들

[input type="text" OOO => OOO에 넣어서 사용!]
  • autofocus : 자동포커싱(입력대기상태)

  • autocomplete : 자동완성

  • placeholder : 입력예시를 보여줌

  • readonly value : 읽기전용

  • disabled : 사용불가
