[ 쿠키 ] : 사용자의 브라우저 내의 쿠키 저장소에 저장
            

[ 세션 ]

-웹브라우저 당 1개의 세션이 자동으로 웹 컨테이너(톰캣서버)에 저장됨
-서버측 메모리에 저장 -> 리스타트하면 내용이 사라짐
-페이지가 바껴도 데이터를 유지시키고 싶을 때(예: 장바구니)

[ 쿠키와 세션 비교 ]

-쿠키는 로그인 전 아이디 저장
-세션은 로그인 후 정보 
  -> 브라우저 당 1개의 세션이 만들어지기 때문에 접속이 되야 세션을 사용할 수 있음

-쿠키는 쿠키객체를 생성하여 사용
 세션은 내장객체
