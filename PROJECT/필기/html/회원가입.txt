[ input type ] 

- placeholder : 입력 도움말
- radio : 객관식 단일선택
- checkbox : 객관식 다중선택
- 라디오 버튼과 체크박스 유의할 점
   > name 속성 값 일치시키기
   > 초기값은 checked(checked="checked")

[ select ] 객관식 펼침목록

<select name=".."> ... </select>

-초기값 : selected="selected"
-2개 이상 선택 : multiple
-n개의 항목이 보이는 상자형 : size="n"
-그룹으로 묶기 : 옵션들을 그룹으로 묶어주고 label 속성으로 제목을 붙인다.
예) <select size="5">
      <optgroup label="오전시간">
       <option>1교시</option>
       <option>2교시</option>
      </optgroup>
     </select> 

[ textarea ] 큰 글상자

속성 -> cols 가로 rows 세로

[ button ] 사용자가 누를 수 있는 버튼

<button type="button" onclick="alert('버튼을 클릭하셨군요!')">

    버튼을 눌러주세요.

</button>

[ fieldset ] 

-from 요소와 관련된 데이터들을 하나로 묶어준다.
-legend : fieldset의 제목, fieldset 안에서만 사용가능

	<h1>필드셋</h1>
	<form action="/examples/media/request.php">
		<fieldset>
			<legend>입력 양식</legend>
			이름 : <br>
			<input type="text" name="username"><br>
			이메일 : <br>
			<input type="text" name="email"><br><br>
			<input type="submit" value="전송">
		</fieldset>
	</form>
