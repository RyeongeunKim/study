img tag로 이미지 넣기

  1. jpg - 트루컬러 표현가능, 실사느낌의 이미지에 적합(상대적으로 파일 용량은 적음)
  2. gif - 최대 256컬러만 표현가능, 단순한 이미지에 적합. 투명배경 지원, 움짤 가능
  3. png - 두가지 형식의 장점 결합. 트루컬러 표현가능. 투명배경 지원.

img의 필수속성

  • alt="..." 이미지 대체텍스트(설명문)
    이미지가 나타나지 않을 때의 설명글
    시각장애인용 음성낭독기에서의 설명글
  • src="이미지경로" 이미지파일의 위치와 정보

원본이미지 크기 그대로 (width와 height를 지정하면 브라우저에 이미지가 더 빨리뜸)

width 속성으로 가로크기(200) 지정

height 속성으로 세로크기(200) 지정

파일의 경로 지정

  • 상대경로 - html문서의 위치를 기준으로 상대적인 경로
    ../*.jpg 상위로의 표시(안 -> 밖 => 한단계 밖의 폴더에 있는 *.jpg)
    a/b/c/*.jpg 하위로의 표시(밖 -> 안 => a폴더 안에 b폴더 안에 c폴더 안에 있는 *.jpg)
    (ex) ../../b/*.jpg 두단계 상위로 이동 후 다시 하위폴더b 안에 있는 *.jpg
  • 절대경로 - 항상 같은 경로
