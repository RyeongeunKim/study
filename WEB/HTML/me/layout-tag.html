layout-tag

html5에서 추가된 레이아웃용 태그들

  • 상단 영역 : 로고, 주메뉴, 로그인 폼 등
    <header> </header>
  • 메뉴 영역
    <nav> </nav>
  • 컨텐츠 중에서 독립형, 완성형 성격을 띄는 영역
    <article> </article>
  • 컨텐츠의 주제별 구분 영역
    <section> </section>
  • 주된 컨텐츠가 아닌 컨텐츠 영역(배너, 바로가기)
    <aside> </aside>
  • 하단영역 : 카피라이트, 연락처 등
    <footer> </footer>
